# 03 최종 검증 및 마감

## 작업 목표

- [x] `web2` lint 상태를 확인하고 남은 정적 검사 오류를 정리한다.
- [x] `storybook` 빌드로 `packages/ui`, `packages/components` 소비 경로를 다시 검증한다.
- [x] 단계 문서를 최종 상태에 맞게 업데이트하고 마감한다.

## 메모

- 2단계 기준 `web2` 빌드는 성공한 상태다.
- 이 단계에서는 검증 위주로 진행하고, 실패 시 필요한 최소 수정만 추가한다.
- `pnpm lint:web2`가 성공했다.
- `pnpm build-storybook`이 다시 성공했다.
- storybook과 web2 빌드 모두 번들 크기 경고는 남아 있지만 실패 원인은 아니다.
