# 02 web2 타입 및 패키지 경계 정리

## 작업 목표

- [x] `web2` 빌드를 막는 타입 오류와 미사용 선언을 정리한다.
- [x] `packages/ui`, `packages/components`에서 노출된 API와 실제 소비 코드의 불일치를 맞춘다.
- [x] `web2`가 현재 `packages` 구현을 정상 소비하는 상태로 맞춘다.

## 메모

- 1단계 검증 기준 실제 실패는 `UserModel` 필드 불일치와 미사용 선언 정리다.
- 이 단계에서는 기능 구조를 다시 흔들지 않고, 현재 마이그레이션 결과를 빌드 가능한 상태로 수습한다.
- `useAuthStore().user`는 `fullName`이 아니라 `name` 필드를 사용하도록 정리했다.
- `packages/components/src/issue-sidebar.tsx`에서 실제로 쓰지 않는 `partPicker`를 제거하고, `isAttachingFiles`는 업로드 버튼/입력 비활성화에 반영했다.
- `packages/ui/src/tiptap-editor.tsx`와 `apps/web2/src/features/issue/components/issue-detail-screen.tsx`의 미사용 선언을 정리했다.
- `pnpm build:web2`가 성공했다.
